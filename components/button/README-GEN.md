@gov.au/buttons
	
---
Buttons signal action. Use them to move the user through the service.

## Install

```bash
yarn add --dev @gov.au/buttons
```
```bash
npm i -D @gov.au/buttons
```

## Usage
```jsx
/*
  Light:  <AUbutton>
  Dark:   <AUbutton dark>
*/
import AUbutton from './../components/button';

<AUbutton>Primary button</AUbutton>
```
